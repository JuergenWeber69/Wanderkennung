# Phase 0 — Zieldefinition & Testdaten

## Genauigkeitsanforderung

- **Wandachse:** ±2 cm
- **Wandstärke:** ±1 cm

Diese Toleranzen sind der Maßstab, an dem die Kalibrierung in Phase 11 beurteilt wird.

## Testdatensatz

- Erste reale Bestandsaufnahme als LAS-Datei vorhanden (Upload steht noch aus).
- Ablage vorgesehen unter `tests/data/` (per `.gitignore` von der Versionierung ausgeschlossen, siehe unten — Rohdaten gehören nicht ins Git-Repo).
- Sobald verfügbar: Referenzmaße (mind. eine bekannte Wandachse + Wandstärke aus der realen Aufnahme) hier ergänzen, damit Phase 11 dagegen prüfen kann.
